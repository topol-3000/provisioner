## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-07 | Tampering | ValkeyStreamsConsumer._dispatch — inbound envelope field | mitigate | Two-phase parse with _RawEnvelope (extra="forbid") + payload model validation; each validation failure is caught as ValidationError → error log + XACK, no state change |
| T-02-08 | Tampering | Unknown envelope.type triggering unexpected handler | mitigate | `payload_class_for()` raises UnknownEnvelopeType for unregistered types → warning log + XACK, no processed_event row; forward-compat per docs/events.md §Evolution rules |
